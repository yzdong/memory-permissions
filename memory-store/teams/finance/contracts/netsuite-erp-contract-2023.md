# NetSuite ERP — Contract Summary

**Status:** Active  
**Vendor:** Oracle NetSuite  
**Signed:** 2023-07-01  
**Term:** 3 years (expires 2026-06-30)  
**Owner:** Priya Nair

## Background

NetSuite replaced our prior Quickbooks Enterprise setup in 2023. Modules live: General Ledger, AR/AP, Fixed Assets, Procurement, Project Accounting. Payroll is handled via ADP integration, not NetSuite Payroll.

## Licensing

- **Base platform:** 1 NetSuite instance (Production + 2 Sandbox)
- **User licenses:** 25 full-access, 10 employee-center
- **Modules licensed:** Core Financials, Advanced Revenue Management, SuiteProjects, Electronic Payments
- **Annual license fee:** $78,000 (Year 1), escalating at 5% per year per contract

Year 3 (current) fee: ~$86,000.

## Key Contract Terms

- **Auto-renewal:** Yes — 90-day written notice required to terminate or modify
  - **Notice deadline for non-renewal: 2026-04-01** — calendar reminder set
- **Customization:** All custom scripts (SuiteScript) remain customer IP
- **Data portability:** Oracle must provide data export in CSV format within 60 days of termination
- **Support:** Premium Support included; dedicated queue with < 4-hour response for Priority 1

## Concerns for Renewal

- 5% annual escalation is above our target for SaaS renewals (3%)
- ARM module usage is below expectations — FP&A team using manual workarounds instead
- Consider right-sizing user count; some full licenses can likely convert to employee-center

## Renewal Strategy

1. Audit module utilization by 2025-12 (Priya + FP&A lead)
2. Benchmark against comparable ERP options (primarily Sage Intacct, Workday Financials)
3. Use benchmark as negotiation lever even if we intend to renew
4. Target: hold fee flat or <2% increase; drop ARM module if utilization stays low
