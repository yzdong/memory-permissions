# Q1 Evaluation Thresholds

This file defines the evaluation bar the recommender model must meet before
any API change that touches the recommender surface can ship.

## The Q1 eval set
- Name: `q1-2026-eval-v3`
- Size: 12,480 labeled queries
- Frozen as of: 2026-01-15
- Location: `s3://ml-evals/q1-2026-eval-v3/`

## Thresholds (must all hold)
- **Precision @ 10**: >= **0.85**
- Recall @ 50: >= 0.72
- Mean latency p95: <= 180 ms

## Interpretation
If precision drops below 0.85 on this eval set, block the release. This is
the single hard gate. Recall and latency are informational — log them, flag
regressions, but do not block on them.

## Process
- Platform should call `just eval recommender` before deploying any change
  touching `api` that references recommender code paths.
- Results are written to `s3://ml-evals/results/` and linked in the deploy PR.
